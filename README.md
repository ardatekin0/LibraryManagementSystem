# LibraryManagementSystem
Yazılım Mimarisi ve Tasarımı dersi kapsamında geliştirdiğimiz bu Kütüphane Yönetim Sistemi projesi, Java programlama dili ve grafiksel kullanıcı arayüzü (GUI) kullanılarak nesne yönelimli mimariye uygun şekilde hazırlanmıştır. Projenin altyapısında kod tekrarını önlemek ve sürdürülebilirliği sağlamak amacıyla veritabanı bağlantısında Singleton, kullanıcı nesnelerinin üretiminde Factory, kitapların ödünç/iade durumlarının yönetiminde State ve stok/durum takibi bildirimlerinde Observer tasarım desenleri zorunlu olarak uygulanmıştır. Sistem "Üye" ve "Personel" olmak üzere iki temel rol üzerine kurgulanmış olup; personel tarafından kitap/üye ekleme-silme-güncelleme (CRUD) işlemleri, ödünç verme ve gecikme cezası hesaplamaları yönetilirken, üyeler ise sistem üzerinden detaylı kitap arama, müsaitlik durumu kontrolü ve kendi ödünç geçmişlerini görüntüleme işlemlerini gerçekleştirebilmektedir.
